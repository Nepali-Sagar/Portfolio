# Sagar Nepali - Portfolio

A modern, responsive personal portfolio website showcasing my projects and skills.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Clean and minimalist interface
- **Fast Performance** - Lightweight and optimized for speed
- **Easy to Customize** - Simple HTML, CSS, and JavaScript structure

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
├── index.html          # Main portfolio page
├── main.js             # JavaScript functionality
├── css/                # Stylesheets
│   ├── style.css       # Main styles
│   └── responsive.css  # Mobile responsive styles
├── assets/             # Images and icons
├── public/             # Static files (CV, etc.)
└── README.md           # Documentation
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information
